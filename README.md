# WXBot - 微信群 AI 机器人

基于 [WeChatFerry](https://github.com/lich0821/WeChatFerry) + DeepSeek/MiMo API + Node.js + SQLite 的微信群聊 AI 机器人。

## 功能

### 🤖 AI 对话
在群里 `@机器人` + 问题，触发单轮 AI 回答（无历史上下文，节省 Token）。支持 DeepSeek 和 MiMo 两种 LLM。

### 🛠 工具
| 命令 | 说明 |
|------|------|
| `天气 <城市>` | 查天气（含逐时预报）|
| `翻译 <文本>` | 中英互译 |
| `新闻` | 热点新闻速览 |
| `历史上的今天` | 今天发生的历史事件 |

### 🎮 娱乐
| 命令 | 说明 |
|------|------|
| `讲个笑话` | AI 讲个冷笑话 |
| `今天运势` | 每日运势 |
| `土味情话` | 随机情话 |
| `抽签` / `抽奖` | 群内随机抽人 |

### 📊 群聊
| 命令 | 说明 |
|------|------|
| `群统计` | 今日发言排行榜、时段分布 |
| `高频词` | 今日高频词汇 TOP15 |
| `今日总结` | 今日群聊 AI 总结 |
| `昨天说了什么` | 昨日群聊回顾 |
| `搜索 <关键词>` | 搜索群聊历史消息 |

### 💰 账户
| 命令 | 说明 |
|------|------|
| `余额` | 查 LLM 账户余额 |
| `用量` | 查 Token 消耗统计 |

### 🎲 游戏
| 命令 | 说明 |
|------|------|
| `猜数字` | 猜 1-100 的数字 |
| `成语接龙` | 成语接龙游戏 |

### ⏰ 定时总结
每天 23:50 自动生成当日群聊总结。

### 🖼 图片发送
支持发送图片到群聊，图片消息跳过随机延迟直接发送。

### 🔍 看门狗自重启
- 事件循环心跳监控（检测 Node.js 卡死）
- 微信连接心跳监控（检测断连）
- 消息队列健康监控（检测积压）
- 连续 3 次失败自动重连或重启
- 连接失败自动重试 5 次，每次间隔 8 秒

### 🌐 Web 管理后台
启动后访问 http://localhost:3080 进入管理界面：
- 服务状态监控（运行时间、内存、连接状态）
- 微信连接管理（群组、联系人、重连）
- 配置在线修改（风控、机器人、LLM）
- 帮助菜单在线编辑（支持自定义模板）
- 实时日志查看（带时间戳）
- 服务重启控制

### 📝 日志增强
- 所有日志输出添加时间戳
- 格式：`[2026/6/18 14:30:00] [WCF] Connected to WeChat`
- API 限流自动重试并记录日志

## 快速开始

### 环境要求
- **操作系统**: Windows 10/11 64位（WeChatFerry 依赖）
- **Node.js**: v18+（推荐 v20 LTS）
- **微信**: 3.9.12.17（[下载](https://github.com/tom-snow/wechat-windows-versions/releases/tag/v3.9.12.17)）

### 部署

```bash
# 克隆
git clone <repo-url> wxbot
cd wxbot

# 安装依赖
npm install

# 配置
cp .env.example .env
# 编辑 .env，填入 LLM_API_KEY

# 运行
npm start
```

详细部署说明请参考 [DEPLOY.md](DEPLOY.md)。

## 技术栈

- **WeChatFerry** — 微信协议接入（DLL 注入）
- **DeepSeek / MiMo** — AI 对话、总结、翻译等
- **SQLite (sql.js)** — 本地消息存储
- **Express** — Web 管理后台
- **Node.js** — 运行环境

## 🧩 凡人修仙传 · 插件系统

支持热插拔的插件系统，当前内置修仙游戏插件。每个插件可独立在群内启用/禁用，并可通过 Web 管理后台管理。

### 📖 新手入门

> 你从一介凡人开始修炼，通过发言积累修为，突破境界，最终飞升化神！

**三步上手：**
1. **发消息** — 每发一条消息自动 +1 修为（道行）
2. **查看境界** — 发送 `我的修为` 看当前境界和进度
3. **突破** — 修为攒够后发 `修炼` 尝试突破到下一境界

### 📋 完整攻略

#### 1️⃣ 修为怎么来（核心）
| 方式 | 获得 | 说明 |
|------|------|------|
| 💬 发言 | +1 修为/条 | 群内每发一条消息自动增加，这是最基础的修行 |
| ✅ 签到 | +1 修为 | 每天发送 `签到`，还有灵石奖励和精力恢复 |
| 🗺 探险 | +3~10 修为 | 发送 `秘境` 探险，有机缘获得修为感悟 |
| ⚔️ 斗法胜利 | +1 修为 | 发送 `斗法` 挑战同群修士，赢了加修为 |

#### 2️⃣ 境界体系（13阶）
发送 `境界大全` 查看完整天梯。从低到高：
```
🧘 练气初期 → 中期 → 后期        (需10~50修为)
🏛 筑基初期 → 中期 → 后期        (需100~500修为)
💎 结丹初期 → 中期 → 后期        (需1000~5000修为)
👁 元婴初期 → 中期 → 后期        (需1万~5万修为)
✨ 化神期·大能                    (需10万修为，圆满！)
```
**越往后突破成功率越低**（90% → 10%），需要用丹药或功法辅助。

#### 3️⃣ 灵石怎么来
| 方式 | 收入 | 说明 |
|------|------|------|
| ✅ 签到 | 每日 1~10+ 灵石 | 连续签到越多奖励越多，法器/灵兽可加成 |
| 🗺 探险 | 每次 5~50 灵石 | 主要收入来源，但也可能亏钱！ |
| ⚔️ 斗法胜利 | +7 灵石 | 赢了赚灵石，输了扣灵石 |

#### 4️⃣ 修炼与突破
- 发送 **`我的修为`** 查看当前境界和下一阶所需修为
- 修为够了发送 **`修炼`** 尝试突破（有成功率）
- 吃丹药再修炼：**`修炼 聚气丹`**（增加成功率 10%~50%）
- 突破失败会掉修为，连续失败增加下次成功率（每次 +5%，上限 30%）

#### 5️⃣ 商店系统
发送 **`商店`** 查看全部商品，支持分类查看：
- **`商店 法器`** — 攻击/防御法宝（提升斗法胜率、签到加成）
- **`商店 灵兽`** — 各种灵兽（签到加成、斗法辅助）
- **`商店 功法`** — 功法秘籍（突破辅助、失败减损）
- **`商店 丹药`** — 丹药列表（修炼辅助）
- **`商店 消耗`** — 符箓·剑阵·傀儡（一次性神器）

**购买用法：** `购买 物品名`（如 `购买 紫凝刃`）
**赠送用法：** `购买 物品名 @对方wxid` 或 `赠送 物品名 @对方wxid`

> 💡 永久物品（法器/灵兽/功法）每人限购 1 件，买前看看自己的储物袋！

#### 6️⃣ 炼丹
去商店买丹药太贵？自己炼！发送 **`炼丹 丹药名`**，只需要商店半价灵石。
可炼制的丹药：聚气丹(10)、筑基丹(50)、凝碧丹(100)、结金丹(200)、天元丹(300)、渡劫丹(500)

#### 7️⃣ 斗法
发送 **`斗法`** 随机匹配同群修士，或 **`斗法 @对方wxid`** 指名挑战。
- 报名费 5 灵石，赢了赚 7 灵石 + 1 修为
- 境界越高胜率越大，克制比自己低境界的有优势
- 装备攻击法宝/灵兽可以提升胜率
- 使用符箓·爆裂符可必定获胜（消耗品）

#### 8️⃣ 秘境探险
发送 **`秘境`** 或 **`探险`** 进入秘境探索（消耗 5 灵石 + 1 精力）：
- 🌿 密林寻药 → 得灵石
- ⛰ 山洞探宝 → 得灵石
- ⚔️ 战妖兽 → 得灵石
- 🏛 上古洞府 → 灵石 + 修为双丰收
- 💊 丹药废墟 → 捡到丹药
- 🎁 救修士 → 获得消耗品
- 💀 毒瘴 → 亏灵石（风险！）
- 🕳 陷阱 → 亏灵石（风险！）

**精力系统：** 每天 20 点精力，签到自动补满，每次探险消耗 1 点。

#### 9️⃣ 储物袋
发送 **`储物袋`** 查看你的全部家当：灵石、修为、已装备的法器/灵兽/功法、丹药数量、消耗品。

#### 🔟 其他命令速查
| 命令 | 说明 |
|------|------|
| `境界大全` / `道途` | 完整境界天梯图（含突破所需修为） |
| `我的修为` / `我的等级` | 查看修仙境界、灵石、装备 |
| `商店` / `珍宝阁` [分类] | 查看可购买物品 |
| `购买 <物品名>` | 购买法器、功法、灵兽等 |
| `赠送 <物品名> @wxid` | 购买并赠送给他人 |
| `储物袋` | 查看灵石、丹药、已装备物品 |
| `灵兽` / `我的灵兽` | 查看拥有的灵兽及效果 |
| `修炼 [丹药名]` | 消耗修为突破当前境界 |
| `炼丹 <丹药名>` | 消耗灵石炼制丹药 |
| `斗法 [@wxid]` | 与人斗法争夺灵石 |
| `秘境` / `探险` | 秘境探险寻宝，消耗精力和灵石 |
| `境界排行` / `境界排行榜` | 境界排行榜 TOP10 |
| `灵石排行` / `灵石榜` | 灵石排行榜 TOP10 |
| `修仙帮助` / `修仙攻略` | 查看本指南 |

### 群隔离

所有插件可针对不同群独立开关。Web 管理后台支持按群设置。

## 风控系统

每小时每群最多 20 次 AI 回复，10% 随机跳过，2-5 秒随机延迟，均为降低被微信风控的概率。支持通过 Web 后台在线调整风控参数。
